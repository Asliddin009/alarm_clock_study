## What this repo is
Flutter app for learning English through alarm-driven tasks and offline study flows.

## Universal Tooling
- Stack: Flutter, Dart
- Package/Build System: Flutter SDK, pub
- Primary Docs: [README.md](/Users/asliddin/projects/alarm_clock_study/README.md), [pubspec.yaml](/Users/asliddin/projects/alarm_clock_study/pubspec.yaml)

## Commands
- Install deps: `flutter pub get`
- Run app: `flutter run`
- Test: `flutter test`
- Lint/Analyze: `flutter analyze`
- Format: `dart format .`

## Project Notes
- Architecture follows feature-first organization under `lib/features`.
- State management uses `bloc`, `flutter_bloc`, and `hydrated_bloc`.
- Localization is enabled with `flutter_localizations` and `intl`.
- Main assets live in `assets/` and are declared in `pubspec.yaml`.
